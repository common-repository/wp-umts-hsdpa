UMTS / HSDPA Verfügbarkeit-Widget

== Description ==

Mit den Widgets von hsdpa-umts-verfuegbarkeit.de können Sie sich die UMTS + HSDPA Multi-Verfügbarkeitsabfrage aller Netzbetreiber direkt auf Ihr Blog holen! Das Beste daran: Sie bieten Ihren Besuchern einen inhaltlichen Mehrwert mit dem Sie gleichzeitig höchst attraktive Affiliate-Provisionen der Netzbetreiber verdienen können (alle Provisionen von Vodafone und E-Plus gehen komplett an Sie, das sind bis zu 100€ pro vermitteltem Kunden).

== Installation ==

Sie MÜSSEN ein Theme verwenden das Widgets ermöglicht. Wenn Sie eine WordPress-Version vor 2.2 benutzen, brauchen Sie außerdem das [sidebar widgets plugin](http://wordpress.org/extend/plugins/widgets/).

1. Entpacken Sie `wp-umts-hsdpa.php` und laden es entweder nach `/wp-content/plugins/widgets/` oder `/wp-content/plugins/` hoch.
1. Aktivieren Sie das Widget durch das 'Plugins' Menü in WordPress.
1. Fügen Sie das neue Widget Ihrer Sidebar hinzu durch das 'Design' oder 'Themes => Widgets' Menü in WordPress.

== Frequently Asked Questions ==

= Welchen Code muss ich in meine Sidebar einbauen? =

Keinen. Dies ist ein Widget. Wenn Sie eine WP-Version älter als v2.2 benutzen, müssen Sie das [widgets plugin](http://wordpress.org/extend/plugins/widgets/) aktiviert haben. Unabhängig von der WP-Version die Sie benutzen müssen Sie ein Theme verwenden das Widgets ermöglicht.
